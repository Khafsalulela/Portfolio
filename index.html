Mobile Designs

Bank Wallets

A clean and secure web interface designed for managing all your bank cards and transaction histories in one place. The design focuses on clarity, trust, and ease of use, allowing users to store multiple card details and view real-time transaction records effortlessly. This project emphasizes data organization, intuitive dashboards, and responsive layouts, combining modern UI/UX principles with a user-centric approach to financial management.

Best Bites

An inviting and user-friendly website designed to teach users how to cook delicious meals step-by-step. Best Bites features visually appealing recipe cards, categorized meal types, instructional videos, and easy-to-follow cooking guides. The design emphasizes accessibility and aesthetics, making learning to cook fun and approachable for all skill levels. This project showcases my ability to combine UI design with content-rich layouts for an engaging and educational user experience.
